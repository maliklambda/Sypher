# Sypher

A query language parser designed for directed property graph databases with a focus on expressiveness and simplicity.

## About

Sypher is a parser for a query language called SyperQL. 
The basic idea is that queries define patterns that are then matched against the graph's internal structure. 

- Declarative syntax — Write queries that describe what to fetch, not how to fetch it.
- Abstraction - Complex low-level operations are described and executed by simple queries.
- Subqueries - Support for recursive subqueries.
